Папка interface - Для вспомогательных функций и моделей. По большей части здесь хранится весь переиспользуемый функционал
Папка pages - Страницы сайта
Папка components - Компоненты
Папка data - Здесь находятся все данные, (переводы языков, фейк данные, большие статические данные)
Папка store - Всё для редакса, если он есть
Папка images - Вся графика
Папка styles - Стили, которые нужны для страниц/вспомогательные стили
Папка ui - UI-Компоненты
Папка assets - Остальные файлы, которые требуются на сайте (видео, 3d-модели и т.д.)
Папка constants - Константы, конфиги
